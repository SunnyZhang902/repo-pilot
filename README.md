# RepoPilot

> Understand any codebase in minutes with AI.

RepoPilot is an AI-powered repository understanding platform that helps developers quickly explore unfamiliar codebases.

🚧 The project is currently under active development.

## Vision

Help developers understand any repository in minutes instead of hours.

## Planned Features

- Repository Import
- Repository Parsing
- Repository Chat
- Architecture Diagram
- Documentation Generation

## Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS

### Backend

- FastAPI
- LangGraph
- PostgreSQL
- Redis
- Tree-sitter
