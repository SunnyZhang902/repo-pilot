# Benchmark Results

Run log for RepoPilot prompt evaluation.

Record each benchmark session here. Do not invent scores — fill in rows only after
a completed evaluation using [evaluation-rubric.md](./evaluation-rubric.md).

---

## How to Record a Run

1. Note the **Prompt Version** (e.g. `v1`) and run date
2. Run summary generation for each repository in [benchmark-repositories.md](./benchmark-repositories.md)
3. Score each output with the rubric
4. Add one row per repository below
5. Summarize findings in [prompt-history/](./prompt-history/) for that version
